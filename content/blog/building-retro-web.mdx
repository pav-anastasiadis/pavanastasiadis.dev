# Building Retro Web Aesthetics

There's something deeply charming about the web of the 1990s. Blinking text. Tiled backgrounds. Beveled buttons that actually look clickable. The web has gotten slicker, but it's lost some soul along the way.

Let's bring some of that soul back.

## The Core Techniques

### Beveled Borders (Win95 Style)

The hallmark of the Windows 95 era was the 3D beveled border effect. It required no images — just clever use of border colors:

```css
/* Classic Win95 raised button effect */
.bevel-raised {
  border: 2px solid;
  border-color: #ffffff #808080 #808080 #ffffff;
  box-shadow:
    inset 1px 1px 0 #c0c0c0,
    inset -1px -1px 0 #404040;
  background-color: #c0c0c0;
}

/* Pressed/sunken state on click */
.bevel-raised:active {
  border-color: #808080 #ffffff #ffffff #808080;
  box-shadow:
    inset 1px 1px 0 #404040,
    inset -1px -1px 0 #c0c0c0;
}

/* CRT scanline overlay */
.crt-screen {
  position: relative;
  background-color: #0a0a0a;
}

.crt-screen::after {
  content: '';
  position: absolute;
  inset: 0;
  background: repeating-linear-gradient(
    to bottom,
    transparent 0px,
    transparent 2px,
    rgba(0, 0, 0, 0.25) 2px,
    rgba(0, 0, 0, 0.25) 4px
  );
  pointer-events: none;
  animation: scanline-scroll 8s linear infinite;
}

/* Neon text glow — because everything needs to glow */
.neon-cyan {
  color: #00ffff;
  text-shadow:
    0 0 5px #00ffff,
    0 0 10px #00ffff,
    0 0 20px #00ffff;
}
```

### The Dark Background

The retro web loved dark backgrounds — not the modern "dark mode" dark, but actual black with a green or white terminal glow. The key is contrast: neon colors pop against near-black.

## The Philosophy

Retro web design isn't about looking broken or dated. It's about tactility. Modern flat design is clean but cold — you can't _feel_ a button. Beveled borders communicate affordance. The glow communicates energy.

The 90s web was alive in a way that today's web often isn't. Let's honor that.

## What This Site Uses

This portfolio uses:

- Press Start 2P for pixel-font headings
- VT323 for terminal-style body text
- Win95-style beveled cards for content containers
- Neon cyan/pink/green accent colors
- CRT scanline overlays for authenticity

The whole thing is built on Next.js + Tailwind v4, proving that retro aesthetics and modern tooling are a perfect pairing.
